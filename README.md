# 仙台高齢化地域の可視化

## Usage

## Purpose

- 仙台の高齢化地域を可視化してみたい
- Python や JavaScript での可視化ツール使用に慣れたい

## Reference

- 仙台市の町・丁目別の年齢別人口データ（令和２年４月１日）
  - 町名別年齢（各歳）別住民基本台帳人口　http://www.city.sendai.jp/chosatoke/shise/toke/jinko/chomebetsu.html
  - データサイズが大きいためこの元ファイルは`.gitignore`

  - データサイズが大きいためこの元ファイルは`.gitignore`
